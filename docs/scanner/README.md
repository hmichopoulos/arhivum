# Scanner Documentation

Documentation specific to the **archivum-scanner** CLI tool.

## Contents

- **REQUIREMENTS.md** - Functional and technical requirements for the scanner
- **DESIGN.md** - Architecture and design decisions for the scanner
- **SCANNER_AGENT.md** - Detailed specification of scanner behavior and algorithms

## Scope

This folder contains documentation about:
- Scanner CLI commands and options
- File walking and hashing logic
- Physical device identification
- Source hierarchy management
- Archive detection and postponement
- Metadata extraction (EXIF, etc.)
- Local storage format (dry-run mode)
- Progress reporting
